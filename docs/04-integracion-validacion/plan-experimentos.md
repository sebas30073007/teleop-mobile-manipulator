## Diseño experimental base

### Variables independientes
- Modalidad de operación (teleoperación XR vs referencia manual en entorno controlado).
- Tipo de tarea logística (traslado, abastecimiento, surtido asistido).
- Complejidad del escenario (baja/media).

### Variables dependientes
- Tiempo de ejecución por tarea.
- Tasa de error operativo.
- Usabilidad percibida.

### Protocolo
1. Preparación del entorno y checklist de seguridad.
2. Ejecución de tareas por condición experimental.
3. Recolección de logs técnicos y cuestionarios de usabilidad.
4. Análisis comparativo y reporte.
